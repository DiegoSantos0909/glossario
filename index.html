Glossário de Tecnologia da Informação

Este glossário apresenta conceitos fundamentais da área de Tecnologia da Informação, auxiliando estudantes e iniciantes a compreenderem os principais termos utilizados no desenvolvimento de software e na informática.

HTML

Linguagem de marcação utilizada para estruturar páginas web.

CSS

Linguagem utilizada para estilizar elementos de páginas web.

JavaScript

Linguagem de programação responsável pela interatividade dos sites.

Bootstrap

Framework CSS que facilita a criação de sites responsivos.

API

Interface que permite a comunicação entre sistemas.

Banco de Dados

Sistema utilizado para armazenar e organizar informações.

Servidor

Computador responsável por fornecer serviços e recursos.

Hospedagem

Serviço que mantém um site disponível na internet.

Domínio

Endereço utilizado para acessar um site na internet.

Responsividade

Capacidade de adaptação do site para diferentes telas.

Framework

Conjunto de ferramentas que facilita o desenvolvimento.

Front-end

Parte visual do sistema com a qual o usuário interage.

Back-end

Parte responsável pelas regras de negócio e processamento.

Git

Sistema de controle de versões utilizado por desenvolvedores.

GitHub

Plataforma online para armazenar e compartilhar projetos Git.
